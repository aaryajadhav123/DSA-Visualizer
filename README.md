AI DSA Visualizer

Overview

AI DSA Visualizer is a full-stack web application that helps students understand Data Structures and Algorithms through step-by-step execution visualization.

Users can enter Java code, and the system will visualize how variables and data structures change during execution. Future versions will include AI-powered explanations, complexity analysis, and support for multiple data structures.
